What is Join?

Join is a Kanban Project Management Tool. With a Tool like this, you can see your tasks and the person who belongs to it.

How to use it

1.

 Add a Task at the Page Add Task.
The Title is required and the Text, everything else is optional

2.

 You can see all the Tasks in the Board
and there you can Edit the single tasks

3.

 In the Summary you have a overall view about the Tasks,
you can see how much tasks you have, todays date and much more
